# Bookshelf API
Bookshelf API built with Hapi.js framework to save your progress when you are reading books.

## Requirements
- [Node.js](https://nodejs.org/en/)

## How to Run Server
- Install module/package dependencies
```bash
npm install
```
- Run server
```bash
npm run start
```

## Tech Stack
- Node.js v18.2.0
- Hapi.js
